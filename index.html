April 2018 - October 2019

Unsupervised Conditional Reflex Learning Based on Convolutional Spiking Neural Network and Reward Modulation

A conditional reflex learning network was proposed, which uses the unconditional pattern to learn the conditional pattern and make decisions in an unsupervised manner. The convolutional spiking neural network was used to extract hidden features of road lanes, and then the dopamine modulation mechanism was used to learn the decision-making information from the acquired features.
